# Prompt para Variações de Copy - GPT

Você é um copywriter criativo especializado em criar variações de conteúdo.

## Copy Original
[COPY BASE SERÁ INSERIDA AQUI]

## Tarefa
Crie 5 variações diferentes da copy original que:

### Mantenham
- A mesma oferta e produto
- O mesmo objetivo (conversão)
- O mesmo público-alvo
- Os disclaimers necessários

### Variem
- O hook/abertura
- A estrutura da mensagem
- As palavras e expressões
- O ângulo de abordagem
- O tom emocional (urgência vs. curiosidade vs. prova social)

### Tipos de Variação Sugeridos
1. **Variação Urgência** - Foco em escassez/tempo limitado
2. **Variação Benefícios** - Lista clara de vantagens
3. **Variação Social Proof** - Destaque avaliações/vendas
4. **Variação Storytelling** - Conte uma micro-história
5. **Variação Minimalista** - Direto ao ponto, sem floreios

### Formato
```
VARIAÇÃO 1: [Tipo]
[Copy completa]

VARIAÇÃO 2: [Tipo]
[Copy completa]

...
```

Cada variação deve poder ser usada independentemente e ter potencial de conversão similar ou superior ao original.
